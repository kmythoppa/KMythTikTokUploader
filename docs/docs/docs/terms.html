서비스 이용약관

본 서비스(KMythAutoUploader)는 TikTok API를 활용하여 인증 기능을 제공하며, 아래 약관에 따라 서비스를 제공합니다.

1. 서비스 목적

이 서비스는 TikTok 콘텐츠 자동 업로드를 위한 도구로, 인증 및 자동화 기능을 제공합니다.

2. 이용자의 책임

  • 이용자는 자신의 TikTok 계정을 본인 명의로 사용해야 합니다.
  • 자동 업로드 설정은 이용자의 책임 하에 관리되며, 본 서비스는 콘텐츠에 대한 책임을 지지 않습니다.

3. 서비스 제공의 중단

기술적 문제 또는 TikTok API 정책 변경 시, 사전 예고 없이 서비스가 중단될 수 있습니다.

4. 저작권 및 콘텐츠 권리

서비스를 통해 업로드되는 콘텐츠의 저작권은 이용자에게 있으며, 본 서비스는 콘텐츠를 저장하거나 수집하지 않습니다.
